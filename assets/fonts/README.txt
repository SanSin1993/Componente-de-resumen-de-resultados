Fuente variable Hanken Grotesk
=============================

Esta descarga contiene Hanken Grotesk como fuentes variables y estáticas.

Hanken Grotesk es una fuente variable con este eje:
  peso

Esto significa que todos los estilos están contenidos en estos archivos:
  HankenGrotesk-VariableFont_wght.ttf

Si su aplicación es totalmente compatible con fuentes variables, ahora puede elegir estilos intermedios
que no están disponibles como fuentes estáticas. No todas las aplicaciones admiten fuentes variables y
en esos casos puedes utilizar los archivos de fuentes estáticas de Hanken Grotesk:
  estático/HankenGrotesk-Medium.ttf
  estático/HankenGrotesk-Bold.ttf
  estático/HankenGrotesk-ExtraBold.ttf

Empezar
-----------

1. Instale los archivos de fuentes que desea utilizar

2. Utilice el selector de fuentes de su aplicación para ver la familia de fuentes y todas las
estilos disponibles
